# website

## Build Setup

```bash
# instala dependencias
$ npm install

# Inicia servidor com hot reaload em localhost:3000
$ npm run dev

# buildar para produção
$ npm run build
$ npm run start

# Gerar projeto estático
$ npm run generate
```

Mais detalhes sobre o nuxt em  [Nuxt.js docs](https://nuxtjs.org).
